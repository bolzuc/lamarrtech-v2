Transelec - Transmisión de Energía

Proyecto: Conversión a SAP S/4HANA y migración desde la nube privada de Entel hacia SAP RISE con Amazon Web Services (AWS).

Alcance: Conversión de SAP ERP 6 a S/4HANA 2022, migración de datos al entorno SAP RISE con AWS, implementación de un SAP WebDispatcher para habilitar el uso de aplicaciones Fiori dentro del entorno RISE.

Descripción del Proyecto

Antes del proyecto, Transelec utilizaba SAP ERP 6 como sistema central para la gestión de su operación. El proyecto consistió en realizar la conversión de SAP ERP 6 a SAP S/4HANA 2022 utilizando un enfoque de "System Move". Este enfoque permitió mantener el sistema fuente operativo durante la conversión. La migración involucró aproximadamente 1 TB de datos al entorno SAP RISE, utilizando instancias EC2 en AWS como puente. Además, se crearon nuevos entornos de desarrollo (DEV) y control de calidad (QAS) a partir de una copia del sistema productivo (PRD) para optimizar la implementación. La coordinación entre SAP y un partner estratégico fue clave para llevar a cabo la migración.

Duración

El proyecto tuvo una duración de 13 meses calendario, abarcando todas las etapas de análisis, conversión, migración, pruebas y estabilización post-Go Live.

Inconvenientes

Durante el proyecto, se encontraron algunos inconvenientes. En la fase de conversión y actualización, fue necesario realizar una remediación de datos transaccionales y financieros para resolver inconsistencias. También se encontraron add-ons de terceros no soportados en la nueva versión de S/4HANA, los cuales requirieron eliminación o ajuste. En la fase de implementación de la infraestructura, se presentaron problemas de conectividad para configurar las comunicaciones desde los diversos sitios operativos de Transelec hacia la nube de RISE, especialmente en interfaces relacionadas con el control de procesos. Además, un dimensionamiento inicial incorrecto provocó problemas de rendimiento durante el Go Live, lo que requirió ajustes posteriores.

Factores de Éxito

A pesar de los desafíos, el proyecto logró el éxito gracias a la colaboración entre SAP, el partner de migración y el equipo interno de Transelec. El uso de instancias EC2 como puente de migración facilitó la transición hacia el entorno RISE. La implementación del SAP WebDispatcher permitió la adopción de Fiori y mejoró la experiencia de usuario.

Resultados

Como resultado del proyecto, Transelec logró la actualización exitosa a S/4HANA 2022, habilitando capacidades avanzadas para la gestión de procesos de transmisión de energía. La consolidación en un entorno moderno y escalable en AWS mejoró la flexibilidad y el tiempo de respuesta del sistema. Finalmente, la optimización del rendimiento del sistema tras ajustes de dimensionamiento aseguró la estabilidad operativa en un entorno de alta demanda.
